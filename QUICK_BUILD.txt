
## When You Update Your App

1. Edit `app_config.ini` - bump the version number
2. Run both build commands
3. Test with new installer

**That's it!** Version automatically syncs to installer.

---

## Common Issues & Fixes

### "ISCC not found"
Install Inno Setup: https://jrsoftware.org/isdl.php

### Build fails with Python errors
Update build dependencies:
```bash
pip install -r requirements_build.txt -U
```

### Installer size too large?
This is normal (60-150 MB with PyInstaller compression)
- Includes Python runtime
- Single-file .exe for users (no Python required)
