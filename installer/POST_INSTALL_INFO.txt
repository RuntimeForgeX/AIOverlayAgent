# AI Overlay Agent – Installation Complete

## Thank You
Thank you for installing **AI Overlay Agent**!

## Next Steps

### 1. Add Your API Key
- Use the **Setup API Key** shortcut created in the Start Menu.
- Open Windows Environment Variables.
- Add one of the following keys:
  - `ANTHROPIC_API_KEY`
  - `OPENAI_API_KEY`
  - `GEMINI_API_KEY`
  - `OPENROUTER_API_KEY`
- ✅ Multiple API provider support

### 3. Keyboard Shortcuts
- **Ctrl + Shift + Space** – Show / hide the app
- **Ctrl + Shift + S** – Capture screen and send to AI
- **Ctrl + Shift + C** – Clear conversation
- **Ctrl + Shift + I** – Focus text input
- **Ctrl + Shift + E** – Export chat to Markdown
- **Ctrl + Shift + Enter** – Send prompt
- **Ctrl + Shift + R** – Regenerate last AI response
- **Ctrl + Shift + X** – Stop AI generation
- **Ctrl + Shift + K** – Copy last AI response
- **Ctrl + Shift + M** – Cycle prompt profiles / toggle mode
- **Ctrl + Shift + Left** – Move UI left
- **Ctrl + Shift + Right** – Move UI right
- **Ctrl + Shift + Alt + Up** – Move UI up
- **Ctrl + Shift + Alt + Down** – Move UI down
- **Ctrl + Shift + Alt + P** – Toggle theme
- **Ctrl + Shift + 0** – Reset window position
- **Ctrl + Shift + T** – Toggle click‑through mode
- **Ctrl + Shift + PageUp** – Jump to previous AI message
- **Ctrl + Shift + PageDown** – Jump to next AI message
- **Ctrl + Shift + Home** – Jump to top of conversation
- **Ctrl + Shift + End** – Jump to latest message
- **Ctrl + Shift + Alt + PageDown** – Cycle model forward
- **Ctrl + Shift + Alt + PageUp** – Cycle model backward
